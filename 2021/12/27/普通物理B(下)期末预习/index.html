考试范围

1 量子物理

1.1 热辐射的定量描述(不考)

  1. 物体辐射总能量及能量按波长分布都决定于温度
  2. 单色辐出度：在物体温度为$T$时，从物体表面单位面积上辐射出的波长介于$\lambda$ 和$\lambda$ +$\mathrm{d} \lambda$ 之间的辐射功率 $\mathrm{d}M$ 与$\mathrm{d} \lambda$的比值：
  1. 辐出度：用M表示，是温度$T$的函数。在一定温度$T$下，物体表面单位面积发射的包含各种波长在内的辐射总功率：
  1. 吸收率：吸收能量与入射总能量的比值
  2. 单色吸收率：当辐射从外界入射到温度为$T$的物体表面时，在$\lambda$ 到$\lambda$ +$\mathrm{d} \lambda$的波段内，吸收能量与入射总能量之比，用$a(\lambda ,T)$表示
  3. 黑体：能够完全吸收外来辐射而没有反射的物体，即$a(\lambda ,T) = 1$
  4. 绝对黑体是一种理想模型，绝对黑体不反射、透射电磁波，但不是自身不发生辐射
  5. 基尔霍夫定律：在同样的温度下，不同的物体或不同表面性质的物体，其单色辐出度与单色吸收率之比是一恒量

  1. 黑体是最理想的发射体

  2. 一个好的吸收体一定也是一个好的发射体

    1.2 普朗克公式 普朗克量子假设

  3. 普朗克量子假设：对于频率为 ν 的谐振子，其辐射能量是不连续的， 只能取某一最小能量的整数倍 $\varepsilon_n = nh\nu$

  4. 普朗克公式

  1. $\varepsilon_n = h\nu$称为能量子，$n$称为量子数

1.3 光电效应

  1. 光电效应：当一束光照 射在金属表面上时，金 属表面会有电子逸出的现象

  2. 逸出的电子称为光电子

  3. 回路中形成电流称为光电流

  4. 存在截止频率（又称红限）$\nu_0$

    • 当入射光的频率$\nu$大于截止频率$\nu_0$时，才能产生光电效应；反之，无论入射光的强度多大，都不能产生光电效应。不同材料的截止频率不同

    • 在入射光频率不变时，饱和光电流随入射光强度$I$增加而增大，即：K阴极逸出的电子数与入射光的强度成正比

    • 当加速电压为零时，光电流并不为零，存在遏止电压 $V\varepsilon$

    • 遏止电压与入射光强度无关，但与入射光的频率成正比
    • 光电子初动能与入射光的强度无关，只与入射光的频率有关

  5. 截止电压反映了光电子的最大初动能

  1. 光电子的动能正比于入射光的频率，而与入射光的振幅（强度）的平方无关。
  2. 当入射光频率$\nu$<红限频率$\nu_0$时，不论光强多大也无光电子逸出。
  3. 不论入射光强度如何，只要入射到金属表面后的$10^{-9}$秒内就能发出光电子。

1.4 爱因斯坦光量子理论

  1. 爱因斯坦光电效应方程
  1. 入射光子的能量：$\varepsilon = h\nu = \frac{hc}{\lambda}$
  2. 逸出功：$W_0 = h\nu_0 = \frac{hc}{\lambda_0}$
  3. 红限：$\nu_0 = \frac{W}{h}$
  4. 打出的光电子初动能：$\frac{1}{2} m v{m}^{2} = eV\varepsilon = h(\nu - \nu_0)$
  5. 产生光电效应的条件光子：$\nu > \nu_0$
  6. 遏止电压：$V_\varepsilon = \frac{h}{e}v - \frac{W}{e}$ ，与入射光的频率成正比，比例系数$\frac{h}{e}$与金属材料的性质无关

1.8 德布罗意波

  1. 自然界是对称统一的。实物粒子和光子一样，也具有波 粒二象性。如果用能量 $E$ 和动量 $p$ 来描述实物粒子的粒子 性，则可用频率 $v$ 和波长 $\lambda$ 来表征实物粒子的波动性。
  2. 德布罗意公式 $\quad E=h v \quad p=\frac{h}{\lambda}$
  3. 注意：实物粒子的波动既不是机械波也不是电磁波，它被称为 “物质波”(matter wave)或“德布罗意波” (de Broglie wave)

1.10 波函数

  1. 1927 波恩(Max Born): 德布罗意波的几率假设 若光波沿 $x$ 方向运动, 则 $E=E_{0} \cos 2 \pi\left(\frac{x}{\lambda}-v t\right)$

  2. 沿 $x$ 方向运动的自由粒子, 其德布罗意波可写为 $\Psi=\Psi{0} \cos 2 \pi\left(\frac{x}{\lambda}-v t\right) \quad$ 或 $\quad \Psi=\Psi{0} e^{i(\vec{k} \cdot \vec{r}-o t)}$

    • $|\vec{k}|=2 \pi / \lambda \quad{\text{波矢; }} \omega=2 \pi v$;

  3. $\Psi$ : 波函数

  4. 爱因斯坦: $|\vec{E}|^{2}$ 一光子密度的几率量度

  5. 玻恩: $|\Psi|^{2}$ 一在给定时间单位空间间隔中发现该粒子的几率
